Generador de Imagen para Redes Sociales

Instrucciones:

  1. Sube tu logo (logo.svg) usando el input de abajo
  2. Se generará una imagen optimizada para WhatsApp, Facebook, etc.
  3. Descarga la imagen haciendo clic derecho y "Guardar imagen como..."
  4. Guárdala como "og-image.png" en la carpeta public/
  5. Actualiza el index.html para usar esta imagen en lugar del logo.svg
